## Projects 🌟

### AutoInsight
[GitHub](https://github.com/arnavsharma09/AutoInsight) — A 7-node multi-agent LangGraph pipeline (Profiler, Router, Planner, Coder, Sandbox, Critic, Reporter) that plans and executes statistical analyses over Polars pipelines, with a human-in-the-loop approval step before code runs. Includes a hybrid rule-based + ChromaDB semantic router across 60+ statistical tests, a Critic agent that auto-corrects methodology based on Shapiro-Wilk normality checks, AST-level sandboxed code execution, and DeepEval-based faithfulness validation. Covered by an 18-test suite at 100% pass rate.

### Credit Risk Prediction System
[GitHub](https://github.com/arnavsharma09/credit-risk-prediction) — A full ML pipeline on 150K+ records addressing 93.7% class imbalance via SMOTE. Benchmarked Random Forest (0.8588 AUC-ROC), XGBoost (0.8492), and Logistic Regression (0.8234); used SHAP for risk-driver explainability and shipped a real-time Streamlit scoring dashboard.

### RetainIQ
[GitHub](https://github.com/arnavsharma09/RetainIQ) — An analytics pipeline on 99,441 orders across 9 relational tables, with a star-schema PostgreSQL model and RFM segmentation across ~95K customers. Ran a pre-committed Mann-Whitney U test (after Shapiro-Wilk confirmed non-normality) isolating delivery time's effect on satisfaction — fast delivery scored significantly higher (4.31 vs 3.75, p<0.001). Shipped a 6-visual Power BI dashboard translating findings into stakeholder recommendations.
